Signatures email

Choisis ta signature, clique Copier, puis colle dans Gmail / Outlook / Apple Mail.

  1. Clique Copier la signature.
  2. Ouvre les paramètres de signature de ton client mail.
  3. Colle (Cmd/Ctrl + V). Les images s'affichent automatiquement.
